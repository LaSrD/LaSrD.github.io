记录： LeetCode上的核心代码题刷多了，遇到标准输入输出的题还是有的吃力，总结一下C++标准输入输出的模板

cin

  • operator >> : 可以从缓冲区连续读取数据，数据以分隔符（空格、tab、换行符）分隔，

    • 若缓冲区中第一个字符是空格、tab或换行这些分隔符时，cin>>会将其忽略并清除，继续读取下一个字符。
    • cin >> 对字符后面残留在缓冲区的 换行符不做处理。
    • >> 表达式返回cin流的状态，输入成功：流保持有效状态 , 条件为真。

  • cin.get() : 读取单个字符

    • 原型：
      • int_type get(); 返回ASCII码值，遇到文件结束符（win Ctrl + z，linux ：Ctrl + d） 返回EOF(- 1)。
      • basic_istream& get( char_type& ch );： 返回对象引用，支持cin.get(b).get(c)
      • ……
    • 从输入缓冲区读取单个字符，不会忽略分隔符。

  • cin.getline() : 读取一串字符串，并以指定的结束符结束

    • 原型：

      basic_istream& getline( char_type* s, std::streamsize count, char_type delim );
// s : 指向要存储字符的字符串的指针
// count : s所指向的字符串的大小
// delim : 结束字符， 被提取不会被存储

    • 分隔符（指定的结束符）有被从缓冲区读取，但 不会 存储到接收字符串的字符地址（与cin.get() 不同）

  • getline()

    • 原型：

      std::basic_istream<CharT,Traits>& getline( std::basic_istream<CharT,Traits>&& input,
                                           std::basic_string<CharT,Traits,Allocator>& str,
                                           CharT delim );
//分隔符字符是从 中提取的，但不会追加到str

    • 和cin.getline()类似，但是cin.getline()属于istream流，而getline()属于string流，是不一样的两个函数

      在遇到需要输入一串带有空格的字符串的时候，要用getline(cin,s),因为 cin>>s，遇到空格就会默认停止。

  • cin.clear()： 将流的所有状态复位

  • cin.ignore() ：从输入流中提取并丢弃字符，直到（包含）结束符

    • 原型：

      basic_istream& ignore（ std：：streamsize count = 1， int_type delim = Traits：：eof（） ）;
//  count
// delim ： 结束符

输入输出练习

OJ输入输出

A+B
1. 输入包括两个正整数a,b(1 <= a, b <= 1000),输入数据包括多组，组数不定。
//输入包括两个正整数a,b(1 <= a, b <= 1000),输入数据包括多组。
//输出 a + b
#include<iostream>
using namespace std;
int main() {
    int a,b;
    while(cin >> a >> b)
        cout << a + b << endl;
    return 0;
}
2. 输入第一行包括一个数据组数t(1 <= t <= 100) ,接下来每行包括两个正整数a,b(1 <= a, b <= 1000)
#include<iostream>
using namespace std;
int main()
{
    int t,a,b;
    cin >> t;
    for(int i = 0; i < t; ++i) {
        cin >> a >> b;
        cout << a + b << endl; 
    }
    return 0;
}
3.输入包括两个正整数a,b(1 <= a, b <= 10^9),输入数据有多组, 如果输入为0 0则结束输入
#include<iostream>
using namespace std;
int main()
{
    int a,b;
    while(cin >> a >> b) {
        if(a == 0 && b == 0)
            break;
        cout << a + b << endl;
    }
    return 0;
}
4.

输入数据包括多组。
每组数据一行,每行的第一个整数为整数的个数n(1 <= n <= 100), n为0的时候结束输入。
接下来n个正整数,即需要求和的每个正整数。

//4 1 2 3 4
#include<iostream>
using namespace std;
int main()
{
    int size;
    while(cin >> size) {
        if(size == 0)
            break;
        int sum = 0;
        int temp;
        for(; size > 0 ; --size) {
            cin >> temp;
            sum += temp;
        }
        cout << sum << endl;
    }
    return 0;
}

接下来t行, 每行一组数据。
每行的第一个整数为整数的个数n(1 <= n <= 100)。

输入数据有多组, 每行表示一组输入数据。

每行不定有n个整数，空格隔开。(1 <= n <= 100)。

#include<iostream>
using namespace std;
int main()
{
    int temp;
    int sum = 0;
    while(cin >> temp) {  //cin>>会将其忽略并清除 空格 ，换行
        sum += temp;
        if(cin.get() == '\n') { // get 来判断到每一行末尾
            cout << sum << endl;
            sum = 0;
        }
    }
    return 0;
}

字符串排序

输入有两行，第一行n
第二行是n个字符串，字符串之间用空格隔开
输出一行排序后的字符串，空格隔开，无结尾空格

#include<iostream>
#include<vector>
#include<string>
#include<algorithm>
using namespace std;
int main()
{
    int size;
    cin >> size;
    cin.ignore(1,'\n');  // 清除缓冲区上的一个换行符
    vector<string> sort_string;
    for(int i = 0; i < size; ++i) {
        string temp;
        getline(cin,temp,' ');  // 以 ’ ‘ 为分隔符提出字符串
        sort_string.push_back(temp);
    }
    sort(sort_string.begin(),sort_string.end());
    for(int i = 0; i < size - 1; ++i)
        cout << sort_string[i] + ' ';
    cout << sort_string[size - 1];
    return 0;
}

多个测试用例，每个测试用例一行。
每行通过空格隔开，有n个字符，n＜100

#include<iostream>
#include<string>
#include<vector>
#include<algorithm>
using namespace std;

int main()
{
    vector<string> vec;
    string temp;
    while(cin >> temp) {
        vec.emplace_back(temp);
        if(cin.get() == '\n') {
            sort(vec.begin(),vec.end());
            for(auto & str : vec) {
                cout << str << " ";
            }
            cout << endl;
            vec.clear();
        }
    }
    return 0;
}

多个测试用例，每个测试用例一行。
每行通过,隔开，有n个字符，n＜100

#include<iostream>
#include<string>
#include<vector>
#include <sstream>
#include<algorithm>
using namespace std;

int main()
{
    vector<string> vec;
    string temp;
    while(cin >> temp) {
        stringstream ss(temp);
        string str;
        while(getline(ss,str,',')) {
            vec.emplace_back(str);
        }
        sort(vec.begin(),vec.end());
        for(int i = 0; i < vec.size() - 1; ++i) {
            cout << vec[i] + ',';
        }
        cout << vec.back() << endl;
        vec.clear();
        ss.str("");
    }
    return 0;
}
